# User Guide: SmartHome Wi-Fi Light Bulb

## 1. Introduction
The SmartHome Wi-Fi Light Bulb allows you to control lighting remotely using a mobile app or voice commands. This guide provides instructions for setup, operation, and troubleshooting.

---

## 2. Product Overview
The SmartHome Light Bulb offers:
- Remote on/off control
- Brightness adjustment
- Color customization
- Voice assistant integration
- Energy-efficient operation

---

## 3. System Requirements
Before you begin, ensure you have:
- A smartphone (Android or iOS)
- Stable Wi-Fi connection (2.4 GHz)
- SmartHome mobile app installed
- Compatible light fixture

---

## 4. Installation

### 4.1 Physical Setup
1. Turn off the power supply.
2. Screw the bulb into the light socket.
3. Turn the power back on.
4. The bulb will blink to indicate pairing mode.

### 4.2 App Setup
1. Open the SmartHome app.
2. Create or log in to your account.
3. Tap **Add Device**.
4. Select **Light Bulb** from the list.
5. Follow on-screen instructions to connect to Wi-Fi.

---

## 5. Using the Product

### 5.1 Basic Controls
- Turn ON/OFF using the app toggle.
- Adjust brightness using the slider.
- Change color from the palette.

### 5.2 Scheduling
1. Go to **Schedule** in the app.
2. Set time and desired lighting state.
3. Save the schedule.

### 5.3 Voice Control
- Link your account with a voice assistant.
- Use commands like:
  - "Turn on the living room light"
  - "Dim the light to 50%"

---

## 6. Troubleshooting

| Issue                  | Possible Cause        | Solution                     |
|----------------------|----------------------|------------------------------|
| Bulb not connecting  | Wrong Wi-Fi band     | Use 2.4 GHz network          |
| Bulb not turning on  | Power issue          | Check switch and socket      |
| App not detecting    | Pairing mode inactive| Reset bulb and retry         |

---

## 7. Resetting the Device
1. Turn the bulb ON and OFF five times quickly.
2. Wait for it to blink.
3. Reconnect using the app.

---

## 8. Safety Information
- Do not use in wet environments.
- Avoid touching the bulb when hot.
- Use only compatible fixtures.

---

## 9. Technical Specifications
- Voltage: 220–240V
- Connectivity: Wi-Fi
- Lifespan: ~15,000 hours
- Supported OS: Android, iOS
